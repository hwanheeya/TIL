## 프록시  
클라이언트로부터 요청을 받아 중계하는 서버. (대리인)  


프록시 개념을 사용하기도 전에, MySQL-Workbench 이용한 것 처럼 ip address와 기타 정보를 입력하니 접근이 가능하다.
